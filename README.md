🌦 Weather App using OpenWeatherMap API

This project is a command-line Weather Application built with Python. It allows users to retrieve real-time weather information for any city by connecting to the OpenWeatherMap API.

The application fetches live weather data and displays:

* Current Temperature
* Feels Like Temperature
* Humidity
* Weather Conditions
* Wind Speed
* Country Information

Technologies Used

* Python
* Requests Library
* REST API
* JSON

Features

* Real-time weather updates
* Easy-to-use command-line interface
* Error handling for invalid city names
* Fast API response processing
* Clean and beginner-friendly code structure

Learning Outcomes

Through this project, I learned:

* How REST APIs work
* Making HTTP requests in Python
* Processing JSON responses
* Handling exceptions effectively
* Building practical real-world Python applications
